# Diabetes Prediction System 🩺

## Project Overview

This project is a **Machine Learning-based Diabetes Prediction System** that predicts whether a patient is likely to have diabetes based on medical and health-related input features.

The goal of this project is to demonstrate how **machine learning models can assist in early detection of diabetes**, which is crucial for timely treatment and prevention of complications.

The system uses **Python** for development, **Scikit-learn** for building the predictive model, and **Streamlit** to create an interactive web interface where users can input patient data and receive predictions.

---

## Features

* Predicts the likelihood of **diabetes in patients**
* Interactive **Streamlit web interface**
* Machine learning model built using **Scikit-learn**
* Simple and user-friendly input system
* Instant prediction results

## Dataset

The model is trained using a **diabetes dataset containing medical diagnostic measurements**. Common features used in the dataset include:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin Level
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

These features are used to predict whether the patient is **diabetic or non-diabetic**.

## Installation

1. Clone the repository

```
git clone https://github.com/praise-ebunoluwa/Diabetes-Prediction-Project.git
```

2. Navigate into the project folder

```
cd Diabetes-Prediction-Project
```

---

## Running the Application

Run the Streamlit app using:

```
streamlit run diabetes.py
```

After running the command, the application will open in your browser where you can enter patient medical data to **predict the likelihood of diabetes**.

---

## Example Workflow

1. Enter patient health information in the Streamlit interface.
2. The system processes the input data.
3. The trained machine learning model analyzes the data.
4. The application displays whether the patient is **likely diabetic or not**.
